== Description ==
 
This plugin transforms the title into a shortcode to repeat for all content. add a shortcode [the_title] wherever you want it to be and be happy.


== Installation ==
 
Add the plugin by uploading the zip file or by installing it from the Wordpress Plugin directory.

Activate the plugin,

use [the_title] in content,

and That's all!

== Frequently Asked Questions ==


1. How to use?
Just add the shortcode [the_title] to the page content, and it will automatically pull the same one used in the page title.

2. Does it work on editing plugins?
Has not been tested at all, but the vast majority have.

3. Are there limits to use?
There are no usage limits, just use responsibly and for best results the search engines involve h1, h2, h3 will work the same way.
